php-training
============

Training project for PHP dev

#1. Setup environment
##On Windows
1. Latest Netbeans
2. Latest Notepad++
3. Virtual machine with Ubuntu/CentOS server (with mysql server, apache/nginx)
4. SVN (And/Or Git) client

##On Linux
1. Latest Netbeans
2. gedit/nano/vi (SublimeText - highly recommend)
3. LAMP/LEMP
4. SVN (And/Or Git) client
